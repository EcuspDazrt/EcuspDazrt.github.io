Featured Research

Network Intrusion
Detection System

Python Random Forest Autoencoder LibPcap SQLite multiprocessing customtkinter CIC-IDS-2017

A real-time network intrusion detection system built end-to-end. LibPcap sniffs raw traffic from the network interface; a flow aggregator groups packets into bidirectional streams and computes behavioral features — inter-arrival times, packet size distributions, byte ratios — that characterize how traffic flows rather than what it contains. An ensemble of a supervised Random Forest classifier and an unsupervised Autoencoder anomaly detector evaluates each completed flow, with flagged events logged to SQLite and surfaced on a live customtkinter dashboard.

// Active Research

Semi-rigorous investigation into model drift poisoning defenses and domain adaptation thresholds. Research focus: quantifying the minimum number of real-environment flows required before adapted model performance stabilizes — experimentally establishing adaptation thresholds across heterogeneous network environments. Architecture decisions formalized as ADRs throughout development.
