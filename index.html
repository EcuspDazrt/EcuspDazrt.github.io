Python ML/AI SQLite LibPcap MultiProcessing

Network Intrusion
Detection System

A real-time network intrusion detection system built end-to-end. LibPcap sniffs raw traffic from the network interface; a flow aggregator groups packets into bidirectional streams and computes behavioral features such as inter-arrival times, packet size distributions, byte ratios, characterizing how traffic flows while avoiding inspecting payloads. A supervised Random Forest classifier and an unsupervised Autoencoder anomaly detector evaluates each completed flow, with flagged events logged to SQLite and surfaced on a live customtkinter dashboard.

// Active Research

Research is currently in progress investigating the system's ability to implement domain adaptation, a process in which the unsupervised model tunes its parameters to adjust to local traffic, and defenses to prevent this process from being poisoned. Details are commited to the repo.
