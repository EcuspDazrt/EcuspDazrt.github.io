15-Puzzle Solver

An A* search implementation with Manhattan distance heuristic that computes optimal move sequences for any valid 15-puzzle state. Solvability is verified before any board is presented. The Python backend is embeded onto the site using pyodide. The frontend animates the solution move-by-move. Demonstrates exploration into informed search, priority queues, and algorithm design fundamentals.

Handwritten Digit Recognizer

Digit classification built from scratch; forward propagation and backpropagation implemented without ML frameworks, trained on MNIST. Draw any digit on the canvas and the model returns a prediction with a confidence score. Demonstrates concrete understanding of the underlying mathematics of how neural networks actually function.
